# sjl - Simple Java Library

SJL is a library which provides a collection of classes and methods to perform specific, reusable tasks, 
such as validations, string manipulations, SQL queries, and more.

## Features

- [sjl-checks](./sjl-checks): provides methods for validating various conditions, such as checking if a string is empty, if a number is positive, etc.

## Why use this instead of other libraries? (e.g. Guava)

No reason at all. This is just a personal project to practice Java and share useful code snippets. If you find it helpful, great! If not, there are plenty of other libraries out there that might suit your needs better.
